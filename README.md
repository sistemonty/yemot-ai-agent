# עוזר קולי AI עם ימות המשיח

עוזר קולי חכם שעובד עם מערכת ימות המשיח, תומך עברית מלא!

## איך זה עובד?

```
מתקשר → ימות המשיח → השרת שלך → Whisper (תמלול) → Claude (AI) → חזרה למתקשר
```

1. מתקשר מגיע לשלוחת API שלך בימות
2. ימות שולחת בקשה לשרת שלך
3. השרת מבקש הקלטה מהמתקשר
4. המתקשר מקליט הודעה קולית
5. **Whisper** מתמלל את ההקלטה לעברית
6. **Claude** מייצר תשובה חכמה
7. התשובה חוזרת למתקשר דרך TTS של ימות

## התקנה

### 1. התקנת הפרויקט

```bash
cd yemot-ai-agent
bun install
```

### 2. הגדרת משתני סביבה

```bash
cp .env.example .env
```

ערוך את `.env`:

```env
# ימות המשיח
YEMOT_SYSTEM=077XXXXXXX
YEMOT_PASSWORD=XXXX

# OpenAI (Whisper לתמלול)
OPENAI_API_KEY=sk-...

# Anthropic (Claude לתשובות)
ANTHROPIC_API_KEY=sk-ant-...
```

### 3. הפעלת השרת

```bash
bun run dev
```

### 4. חשיפה לאינטרנט

השרת צריך להיות נגיש מהאינטרנט. אפשרויות:

**אפשרות א: LocalTunnel (לבדיקות)**
```bash
bun run tunnel
```

**אפשרות ב: ngrok**
```bash
ngrok http 3000
```

**אפשרות ג: שרת אמיתי**
העלה לשרת VPS או Vercel/Railway

### 5. הגדרת שלוחת API בימות המשיח

1. היכנס לניהול המערכת: https://www.call2all.co.il/
2. צור שלוחה חדשה מסוג **"מודול API"**
3. הגדר את כתובת השרת:
   ```
   https://YOUR_DOMAIN/yemot
   ```
4. Method: POST

## בדיקה

התקשר למערכת שלך בימות והגע לשלוחת ה-API.
העוזר ישאל אותך:
1. מה השם הפרטי שלך?
2. מה שם המשפחה?
3. מה מספר תעודת הזהות?
4. כמה ילדים יש לך?

## התאמה אישית

### שינוי השאלות

ערוך את `SYSTEM_PROMPT` בקובץ `src/server.ts`:

```typescript
const SYSTEM_PROMPT = `אתה עוזר קולי של חברת XYZ.
המשימה שלך: לענות על שאלות לקוחות על המוצרים שלנו.
...`;
```

### הוספת מאגר מידע

אפשר לחבר Airtable, Google Sheets או כל מקור מידע אחר.

## מבנה הפרויקט

```
yemot-ai-agent/
├── src/
│   ├── server.ts    # שרת Webhook ראשי
│   ├── whisper.ts   # תמלול עם OpenAI Whisper
│   ├── claude.ts    # תשובות עם Claude
│   └── tts.ts       # Text-to-Speech (אופציונלי)
├── .env             # משתני סביבה
└── package.json
```
